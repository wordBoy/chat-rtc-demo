Accessible Chat Demos

This is an interactive example set for our CSUN 2017 presentation AT&T Mobile and Desktop Chat with ARIA: A Look Under the Hood. If you've not heard of that presentation, that is not a problem. This site is about learning some fundamentals around web accessibility involving ARIA, modal windows and more, using AT&T's interactive text chat as an inspiration for our examples.

This demo is responsive, and intended to be viewed and explored both in mobile and desktop environments.
