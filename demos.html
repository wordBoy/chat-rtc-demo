AT&T Chat Primer

Chat demos are the different parts of chat, also called the "taxonomy". To learn more, we highly recommend you spend some time on the att.com website. The Help & Support page has a chat instance - just give it a minute to appear once the page renders. Select each image below to see more detail.

Chat Launch Button

Shockingly, the chat launch button initiates chat when operated by the user. It is also called the Point of Invocation (POI) or Anchor Button.

Chat Launch Button Demos

Touch Guide

When you launch chat, you usually need to clarify what you need to chat with someone about. This is where the Touch Guide comes in, helping the chat user get to as specific an area as possible so they get the most help possible, as fast as possible.

Touch Guide Demo

Pre-Chat Survey

Before the chat session itself, it is sometimes useful to ask the user who wants to chat some questions. For example, their first name.

Pre-Chat Survey Demo

The chat window is where the chat takes place.
