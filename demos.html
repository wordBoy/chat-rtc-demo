AT&T Chat Primer

Welcome! This is an online primer for our CSUN 2017 presentation AT&T Mobile and Desktop Chat with ARIA: A Look Under the Hood. If you've not heard of that presentation, that is not a problem. This site is about learning some fundamentals around web application development involving ARIA, modal windows and more. We use AT&T's interactive text chat solution as the framework by which we explore all these features.

Chat Demos

Chat demos are the different parts of chat, also called the "taxonomy". We'll provide a brief explanation of what each part is to get you oriented. To help you decide where to start, each demo has sub-bullets with topics that we'll explore in the code.
